# Refactor Skill

Identifies and applies safe refactoring patterns for the CentralStore codebase.

## Trigger
When user asks to refactor, clean up, or improve code quality.

## Process
1. Analyze target files, identify code smells
2. Plan refactoring with risk assessment
3. Apply one change at a time, build+test after each
4. Special attention to module boundaries and MediatR patterns
